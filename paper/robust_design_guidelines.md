# Step 8 — 신뢰구간 포함 강건 설계 지침 (자동 생성)

방법: 캘리브레이션 관통 Monte-Carlo (N=500) — Table 1 측정노이즈(σ=0.08 dec), 디지타이즈 앵커(σ=0.10 dec), placeholder 물성(D_par, S_par, Ea, r_pin), M_crit(×/÷3), 피로상수(σ_c0, m)를 표본화하고 **매 표본마다 Stage A/B 재캘리브레이션** → 파라미터 상관(f_res↔P_par 등) 자동 보존.

**G1 [HIGH]** 무기층 하한: 핵 폐색 완료 두께 d_in ≥ 22 nm (95% CI 21–26 nm)  
  메커니즘: f_pin 하강분지; Table1 15→20nm 급락으로 구속 | 유효조건: Wu Table1

**G2 [HIGH]** 무기층 상한: 균열 개시 d_in < 44 nm (95% CI 38–48 nm) — 초과 시 수명·내구성 동시 붕괴(지배 구역)  
  메커니즘: f_crack 재개방 + σ_c(d) 저하 | 유효조건: 유연기판 취급변형 레짐

**G3 [LOW-MED]** 파레토 구간(G1–G2 사이)에서 T80↔내구성 교환율: N_f ∝ d^-3.6 (95% CI -4.9–-0.0); 10년 내구성 상한 d_in ≤ 30 nm (95% CI 16–49 nm) — 단, 피로 프록시 사전분포 하에서 어떤 두께로도 10년 미달일 확률 16% (모델 한계의 정직한 정량화)  
  메커니즘: Basquin 프록시(sigma_c0, m 사전분포) | 유효조건: 내구성 모델은 지수 프록시 — 1차 문헌 보강 대상

**G4 [HIGH(구조)/MED(절대값)]** 쌍수 법칙: 선형·무포화 구조는 강건(모든 MC에서 R² ≥ 0.9998); 상대 한계이득 a/T80(n=3) = 0.46 (95% CI 0.41–0.49)/pair. 절대이득 +7592 h (95% CI 1321–45276 h)/pair의 넓은 CI는 M_crit ×/÷3 사전분포가 지배 → 절대수명은 M_crit 캘리브레이션 대기, 구조 결론은 확정  
  메커니즘: R_tot 가산 구조 | 유효조건: n≥2

**G5 [MED]** 유기층: 공정 최소 두께(≥100 nm) 권장 — n≥2에서 R_sand ∝ 1/d_org (측면 병목 활성화); n=1에선 무영향  
  메커니즘: Step7 활성화 사슬; d_org=100은 경계해 | 유효조건: 핀홀 복제 미모델링 → 하한은 공정 제약

**G6 [HIGH]** 열·광학은 이 재료계에서 기하 자유도 아님(스팬 0.10 K / 0.35 %p) → 두 목적은 재료 선택 단계로 이관  
  메커니즘: 전도저항 mK 수준 + 굴절률 정합 정량화 | 유효조건: Al2O3/parylene C

**G7 [HIGH]** 강건성: M_crit ×/÷3 불확실성에도 임계값·순위 불변 — 절대 T80만 스케일 (KNEE 설계 T80 = 22688 h (95% CI 3735–128007 h))  
  메커니즘: 캘리브레이션 관통 MC가 상관 보존 | 유효조건: 지침은 절대수명이 아닌 구조 결론

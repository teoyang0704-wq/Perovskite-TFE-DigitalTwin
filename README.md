# Perovskite-TFE-DigitalTwin
Physics-informed digital-twin framework for thin-film encapsulation (TFE) geometry design:
literature-calibrated defect-mediated moisture physics, exhaustive Pareto exploration,
exact Shapley/Sobol explanation, and calibration-through uncertainty quantification.

**Live demo:** https://tfe-twin.streamlit.app  (design + environment sliders, Pareto map, constraint optimizer)

Full paper draft, SI, provenance database and analysis scripts ship in the release bundle
(v0.9.2); this repository state is the app-runtime core. MIT License. (c) 2026 Teo Yang.
